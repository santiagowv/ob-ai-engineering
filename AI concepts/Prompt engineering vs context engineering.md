# Prompt engineering
It <span style="color:rgb(216, 203, 251)">instructs the LLM</span> to behave in a certain way.
# Context engineering
<span style="color:rgb(216, 203, 251)">Providing the right context</span> for what we want the LLM to do.
- Good prompt + poor context = poor result.
## Context poisoning
Generates <span style="color:rgb(216, 203, 251)">poor reasoning</span>.
- <span style="color:rgb(216, 203, 251)">Outdated</span> information.
- <span style="color:rgb(216, 203, 251)">Bad</span> information.
## Context distraction
Generates <span style="color:rgb(216, 203, 251)">weaker results</span>.
- Too much <span style="color:rgb(216, 203, 251)">irrelevant information</span> mix into the context.
## Context clashing
Generates <span style="color:rgb(216, 203, 251)">inconsistent behavior</span>.
- <span style="color:rgb(216, 203, 251)">Contradicting</span> information.
