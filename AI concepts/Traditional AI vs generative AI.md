# Stakeholders role
## Business leaders and decision-makers
Approve the selection when it's <span style="color:rgb(216, 203, 251)">aligned with the business priorities</span>.
## Product owners
Might require influence or <span style="color:rgb(216, 203, 251)">have more control of the model behavior</span> to align it with the product priorities.
## Domain experts
Apply their domain expertise to <span style="color:rgb(216, 203, 251)">improve model effectiveness</span>.
## End users
<span style="color:rgb(216, 203, 251)">Understand the output of the model</span>, and how to incorporate the output for more informed decision-making.
# When to combine generative AI with traditional AI
We can <span style="color:rgb(216, 203, 251)">use output from a traditional AI model as part of the prompt for a generative AI</span> model.

| Traditional AI                                                                                                                                                                                               | Generative AI                                                                                                                                                               |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Historical data to <span style="color:rgb(216, 203, 251)">forecast customer churn probability</span>                                                                                                         | Integrate with an LLM or generative AI-powered chatbot to <span style="color:rgb(216, 203, 251)">explore the predictions by using natural language conversations</span>     |
| <span style="color:rgb(216, 203, 251)">Forecast risks</span> of a specific use case                                                                                                                          | <span style="color:rgb(216, 203, 251)">Simulate different scenarios</span> to help in formulating possible mitigation strategies                                            |
| <span style="color:rgb(216, 203, 251)">Identify customer segments</span> to help create personalized marketing and campaign creation                                                                         | <span style="color:rgb(216, 203, 251)">Generate personalized marketing content</span> tailored to each identified segment                                                   |
| <span style="color:rgb(216, 203, 251)">Computer vision can detect and classify sign language</span> to translate video input into text                                                                       | AI can <span style="color:rgb(216, 203, 251)">add understanding of context and nuance within sign laguage</span>, allowing for more optimized translation into written text |
| Perform video analytics an use video intelligence capabilities to <span style="color:rgb(216, 203, 251)">perform object detection, person detection, text detection, and extraction from video assets</span> | Create novel experiences like <span style="color:rgb(216, 203, 251)">chatbots, listings, reports, or articles</span>                                                        |
